This project contains a collection of Artificial Intelligence laboratory programs implemented using Python and C. These programs demonstrate important AI concepts such as search algorithms, human verification systems, and intelligent behavior simulation.

The main objective of these programs is to understand how AI techniques are used for problem solving, decision making, and distinguishing between human and machine behavior.


1. Turing Test Simulation

This program demonstrates a simple version of the Turing Test, where a judge communicates with both a human and a machine. The identities of the participants are hidden, and the judge must determine which participant is the human based on their responses.

Key Concepts

-> Intelligent behavior simulation

-> Human vs machine interaction

-> Decision evaluation


2. CAPTCHA Authentication System

This program generates a random CAPTCHA consisting of letters and numbers. The user must correctly enter the CAPTCHA to verify that they are a human user.

Key Concepts

-> Random string generation

-> Human verification techniques

-> Basic security implementation


3. Breadth First Search (BFS)

This program implements the Breadth First Search algorithm to explore nodes level by level until the goal state is found.

Key Concepts

-> Graph searching

-> Queue data structure

-> Systematic node exploration


4. Depth First Search (DFS)

This program demonstrates the Depth First Search algorithm, where the search explores one path deeply before backtracking to explore alternative paths.

Key Concepts

-> Graph traversal methods

-> Stack-based implementation

-> Backtracking strategy


5. Depth Limited Search (DLS)

Depth Limited Search is a variation of DFS where the search depth is restricted to a specified limit to prevent infinite exploration.

Key Concepts

-> Controlled depth searching

-> Recursive algorithms

-> Graph exploration


6. Iterative Deepening Depth First Search (IDDFS)

This algorithm combines the advantages of BFS and DFS. It repeatedly performs Depth Limited Search while gradually increasing the depth limit until the goal is found.

Key Concepts

-> Iterative searching approach

-> Depth expansion strategy

-> Memory-efficient searching

7. Missionaries and Cannibals Problem using BFS

This program solves the Missionaries and Cannibals problem using the Breadth First Search algorithm. In this problem, three missionaries and three cannibals must cross a river using a boat that can carry at most two people. The constraint is that cannibals must never outnumber missionaries on either side of the river.

The BFS algorithm explores possible safe states and finds an optimal sequence of moves that safely transfers everyone to the other side.

Key Concepts

-> State space representation

-> BFS problem solving

-> Constraint checking and safe states
